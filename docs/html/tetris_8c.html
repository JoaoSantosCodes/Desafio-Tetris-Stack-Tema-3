◆ inicializarSistemaExpert()

void inicializarSistemaExpert ( SistemaExpert * sistemaPtr)

Inicializa o sistema Expert com valores padrão.

Esta função configura todos os parâmetros iniciais do sistema de pontuação e estatísticas avançadas do Nível Expert.

Parâmetros
sistemaPtrPonteiro para a estrutura SistemaExpert a ser inicializada

Valores iniciais configurados:

  • Pontuação: 0 pontos, multiplicador 1x, nível 1
  • Combos: todos zerados, sem tipo anterior
  • Dificuldade: nível 1, fator 1.0, 1000 pontos para próximo nível
  • Estatísticas: todos contadores zerados
  • Conquistas: nenhuma desbloqueada
